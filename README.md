# react-native-alium

A React Native wrapper for Alium Android SDK

## Installation

```sh
npm install react-native-alium
```

## Usage

```js
import { configure, showSurvey } from 'react-native-alium';

// ...

configure(url);
loadAliumSurvey(current_screen);
```
